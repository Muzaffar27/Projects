How to run 
==========
1. Type "make" to compile

2. Info:
	./main image info
	Ex: ./main comp3430-good.img info
3. List
	./main image list
	Ex: ./main comp3430-good.img list
4. Get
	./main image get fileTofind(the name of the file only)
	Ex: ./main comp3430-good.img get pandp.txt
	Note:   We are told "Provide a command in your readme that works for a get for you." 
		Mine works when you just provide the name of the file to get.
		The name of the file to search should work if it is in lower case also.
To clean
========

Use "make clean" to clean.
