Quetion 1 Elf File
==================

This is a program that takes a file and prints it's header(elf,program and section).

To run this program: 
./question1 helloout64.txt

Question 2 Process
==================

This is the process hearder. Create n processes from config text file. And then delete
or create more depending on change in config file.

To run this program: 
./question2 config.txt

Question 3 Thread
==================

This is the thread hearder. Create n threads from config text file. And then delete
or create more depending on change in config file.

To run this program: 
./question3 config.txt

Compiling for all questions:
----------------------------

Run 'make' in the respective question directory to compile the code

Run 'make clean' to clean.
